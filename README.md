# react-clock

A simple React clock project with **multilingual support** and **dark/light mode toggling**.

## Requirements
- Node.js (recommended latest LTS)
- Yarn

## Run (development)
yarn run start

## Build (production)
yarn run build

## Features
- **Multilingual support**: switch the app language. ___(with more coming!)___
- **Dark/Light mode toggling**: switch between themes.
